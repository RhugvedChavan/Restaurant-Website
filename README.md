# Restaurant-Website : - 


Ember & Oak is a fully responsive restaurant website built with vanilla HTML, CSS, and JavaScript without relying on external frameworks. The website features an attractive landing page, interactive menu tabs, image gallery, testimonial carousel, reservation form with client-side validation, Google Maps integration, newsletter subscription, and responsive navigation. It is lightweight, easy to customize, and can be deployed without any build process or backend server.


# Objective :-


Develop a modern, responsive, and user-friendly restaurant website that provides visitors with an engaging online experience. The project showcases the restaurant's menu, story, gallery, customer testimonials, reservation system, location details, and contact information while ensuring compatibility across desktop, tablet, and mobile devices using only HTML, CSS, and JavaScript.


# Key Features :- 

1. Responsive Design optimized for desktop, tablet, and mobile devices.
2. Interactive Restaurant Menu with categorized tab navigation.
3. Reservation Form featuring real-time client-side validation.
4. Customer Testimonial Carousel with automatic sliding.
5. Gallery, Location Map, Newsletter Signup, Sticky Navigation, and Scroll Animations for an enhanced user experience.


# Result :- 


The project successfully delivers a complete restaurant website that is visually appealing, responsive, and interactive. Users can browse the menu, explore restaurant information, view the gallery, make reservation requests, and locate the restaurant through an embedded Google Map. The website operates entirely on the client side and can be launched directly in a web browser without additional setup.


# Conclusion :- 


Ember & Oak demonstrates how a professional restaurant website can be developed using only core web technologies. The project emphasizes clean design, responsive layouts, accessibility, and interactive user interfaces while maintaining simplicity, fast performance, and easy maintainability. It serves as a strong foundation for future enhancements such as backend reservation management, online ordering, authentication, and payment integration.
